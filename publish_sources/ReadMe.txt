ダウンロードしていただきありがとう御座います。

■ソフト情報
ソフト名:WirePlanet
バージョン:1.1.2
最終更新:2017/12/20

■動作環境
OS:WindowsVista以降の32/64bit WindowsOS
物理メモリ:100MB以上の空きが有ること
その他:DirectX9以上がインストールされていること。シェーダーモデル2.0以上に対応しているグラフィックスデバイスの搭載を推奨
※テスト環境はWindows10 v1709 Pro 64bit

■説明
　線だけで出来た、シンプルなシューティングゲームです。メニューでの操作はCで決定、Aでキャンセルとなります。ゲームでの細かい操作はマニュアルを御覧ください。30分ほどでクリアできますが、難易度は高めです。
　マニュアルは、ゲーム本体(WirePlanet.exe)のある場所のManualフォルダ内にHTML形式で保存されています。Manualフォルダ内のindex.htmlを開いてください。ゲーム内のタイトルメニューから「マニュアル」を選択しても開くことができます。マニュアルには攻略のコツも記載されているので、詰まったら見てみてください。

■インストール方法など
インストール方法:解凍後にそのままプレイできます。
アンインストール方法:レジストリなどに書き込みは行っておりません。フォルダごと削除してください。
起動方法:WirePlanet.exeを起動すれば始められます。
バージョンアップ方法:ファイルを再ダウンロードしてください。
セーブデータのバックアップ・移動:WirePlanet.exeが存在するフォルダ内のsave_data.savがセーブデータ本体です。このファイルをコピーして別の場所に保管しておけばバックアップを取ることができます。また、このファイルを移動することでセーブデータの移動も行えます。
